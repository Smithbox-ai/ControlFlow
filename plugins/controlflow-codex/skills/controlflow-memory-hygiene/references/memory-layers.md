# Memory Layers

## Session Scratch

- Temporary notes for the current conversation
- Safe for rough hypotheses and in-progress reasoning
- Can be dropped freely

## Task Artifact

- Plan files, implementation notes, or completion summaries tied to one task
- Holds the history that would be noisy in a global note

## Repo-Persistent Note

- Current objective
- Active phase
- Durable facts worth carrying across tasks
- Blockers that still matter right now

Only the third layer should survive as a general repo memory.
